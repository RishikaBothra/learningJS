# ===

```javascript
let a = 1       // integer
let b = "1"     // string

if (a === b) {
    console.log("same");
} else {
    console.log("not so same");
}
```

// b is the string boy here so what actually happens is ki ahhh... a ka jo data type hai wo b ka data type mai convert ho jata hai, so basically both a and b becomes string over here
